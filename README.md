# flask-product-app
This project displays products from the DummyJSON API in a searchable and paginated table.

Features:
* Product search
* Pagination
* Product details
* Product thumbnails
* Expandable image gallery
* Basic error handling

Technologies:
* Python
* Flask
* Jinja
* HTML
* CSS
* JavaScript
* Git for version control

How It Works:
The Flask backend fetches product data from the DummyJSON API.
The data is passed to a Jinja template, which displays the product table.
Search and pagination use URL query parameters.
JavaScript is used to open and close each product’s image gallery.
